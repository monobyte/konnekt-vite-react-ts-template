## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Project Structure

- `src/` - Application source code.
- `index.html` - Entry HTML file.
- `tailwind.config.js` - Tailwind CSS configuration.
- `vite.config.ts` - Vite configuration.
